翻轉教室


在過去，我們在網路上的討論區，常常看到同學們在學校已經學會統計、學會用R 做分析，但是卻不熟悉R 本身，導致在學習分析套件，或是在實務的資料整理上，有諸多問題與挫折。

我們也理解這類R 的基礎知識，是不容易在課堂上以傳統教學方式教授的。因此，我們運用R 的swirl套件，開發了這套教材，讓學生能在R 中學R，並且在學習中給予學生挑戰，增強學生的動機。

除了基礎的R 知識之外，我們也以實際的開放資料為例，介紹處理開放資料的方法，以及處理中文資料的技巧。台灣是2015年全世界做開放資料最好的國家，而R 絕對是處理這些開放資料的利器。我們希望學生能善用政府釋出的開放資料。

本教材以創用CC 3.0 姓名標示的方式釋出，所以老師們可以自由地把本教材運用在課程之中。我們建議老師們將本教材當成的補充作業使用，讓同學在學R 做分析之餘，也可以學習R本身的知識和處理開放資料的技能。如果有需要，我們也願意到課堂上給一節課程長度的介紹，讓同學知道如何使用本教材。有興趣的老師很歡迎來信聯繫我們。


獨立性

老師不需要再煩惱教R 的事情，可以給我們一到兩小時介紹R 的背景知識，剩下的時間交由學生做作業自學。

開放

R Basic 交給我們！開放的教材讓老師能夠以最低的成本取得教材，使老師能聚焦於R 基礎使用、Data 的 Engineering 與 Visualization 。教材本身的獨立性，使老師們專注在課程的核心。

效果

學生經過這樣的練習之後，真的能學到紮實的 R 基礎，繼續學後續「老師們傳授的核心知識」
